# Engine Core

The Engine Core is the foundation of the offline-first Cinematic AI Video Studio.

## Responsibilities
- Project state management
- Timeline orchestration
- Clip lifecycle
- Playback & preview
- Non-destructive editing

## Core Modules

### Timeline Engine
- Multi-track timeline
- Frame-accurate timing
- Keyframe support

### Clip System
- Video
- Audio
- Image
- Text
- AI-generated assets

### Playback Engine
- Real-time preview
- Low-resolution proxy mode
- Scrubbing & seeking

### State Management
- Undo / Redo
- Autosave
- Crash recovery

## Offline Guarantee
All engine functionality works without internet access.
