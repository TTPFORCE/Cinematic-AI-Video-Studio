# Upgrade Flow System

Defines progression logic as a game-style unlock journey.
This is not pricing — this is experience design.

---

## Core Principles

- Progression must feel earned
- Power is revealed, not dumped
- Obsidian is never advertised
- Users discover depth gradually

---

## Progression Stages

### Stage 1 — Bronze (Entry)
Unlocks:
- Basic timeline
- Limited tracks
- Standard export

UX:
- Clean
- Fast
- Friendly

Message:
“Your journey begins.”

---

### Stage 2 — Silver (Momentum)
Unlocks:
- More tracks
- Better previews
- Faster execution

UX:
- Confidence
- Speed

Message:
“Momentum unlocked.”

---

### Stage 3 — Gold (Precision)
Unlocks:
- Advanced keyframes
- Precision controls
- Better color tools

UX:
- Professional
- Focused

Message:
“Precision meets power.”

---

### Stage 4 — Platinum (Mastery)
Unlocks:
- Parallel execution
- Higher quality exports
- Advanced pipelines

UX:
- Command
- Authority

Message:
“You now command the system.”

---

### Stage 5 — Obsidian (Legend)
Unlocks:
- Nested timelines
- Distributed render
- Experimental systems

UX:
- Minimal
- Silent
- Powerful

Message:
“You were never meant to ask.”

---

## Upgrade Discovery Rules

- UI suggests only the *next* stage
- No full tier list shown by default
- Obsidian never appears in pricing
- Obsidian may appear in:
  - Lore
  - Achievements
  - System messages

---

## Anti-Abuse Rules

- No downgrade prompts
- No forced upgrades
- No nagging UI
