# Upgrade Engine

Centralized system controlling feature access, limits, and unlock progression.

## Design Principles
- Offline-first (no server dependency)
- Deterministic feature gating
- No feature degradation on downgrade
- Obsidian tier is invitation-only

## Upgrade Tiers

### Bronze (Default)
- Basic timeline (2 tracks)
- Standard effects
- 1080p export
- Watermark enabled

### Silver
- Extended timeline (5 tracks)
- Advanced transitions
- 4K preview
- No watermark

### Gold
- Unlimited tracks
- Advanced keyframes
- Color grading tools
- Priority render queue

### Platinum
- GPU execution control
- High bit-depth rendering
