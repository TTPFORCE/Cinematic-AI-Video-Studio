# Timeline System

Core time-based composition model.

## Concepts
- Timeline: global time context
- Track: ordered lane of clips
- Clip: media segment with start/end
- Keyframe: parameter value at time

## Rules
- Deterministic playback
- Frame-accurate positioning
- No cloud dependency

## Upgrade Hooks
- Max tracks (tier-based)
- Advanced keyframes (Gold+)
- Nested timelines (Obsidian)
